All notable changes to JEngine v1.1.x are documented here.

For the complete changelog including older versions, see the [GitHub CHANGE.md](https://github.com/JasonXuDeveloper/JEngine/blob/master/CHANGE.md).

<Accordions>
<Accordion title="v1.1.5 (February 1, 2026)">
**New Features:**
- Added `JContainer` component with responsive max-width constraints following Tailwind/Bootstrap patterns ([#613](https://github.com/JasonXuDeveloper/JEngine/pull/613))

**Packages:** Core v1.1.5, Util v1.1.3, UI v1.1.3
</Accordion>

<Accordion title="v1.1.4 (January 30, 2026)">
**Bug Fixes:**
- Fixed false positive error detection in `BuildManager` — reset error baseline after filtering YooAsset catalog errors during `GeneratePolymorphicCodes` step ([#610](https://github.com/JasonXuDeveloper/JEngine/pull/610))
</Accordion>

<Accordion title="v1.1.3 (January 30, 2026)">
**New Features:**
- Added JAction parallel execution with `JActionExecution`, `JActionExecutionHandle`, and task snapshot isolation ([#607](https://github.com/JasonXuDeveloper/JEngine/pull/607))

**Packages:** Core v1.1.3, Util v1.1.3, UI v1.1.3
</Accordion>

<Accordion title="v1.1.0 (January 29, 2026)">
**New Features:**
- **New package: `jengine.ui`** — editor UI component library with monochrome design system ([#592](https://github.com/JasonXuDeveloper/JEngine/pull/592))
- Extracted MessageBox from core to `jengine.ui` package with new `JEngine.UI` namespace ([#590](https://github.com/JasonXuDeveloper/JEngine/pull/590))
- Added `Prompt` abstraction layer — delegate-based dialog decoupled from core ([#590](https://github.com/JasonXuDeveloper/JEngine/pull/590))
- Added Claude Code plugin for AI-assisted development ([#598](https://github.com/JasonXuDeveloper/JEngine/pull/598))
- Improved theming system with cursor support and hover effects ([#596](https://github.com/JasonXuDeveloper/JEngine/pull/596))

**Bug Fixes:**
- Fixed CS0311 compilation error — converted `AsyncOperation` to `UniTask` for Unity 6 compatibility ([#595](https://github.com/JasonXuDeveloper/JEngine/pull/595))

**Breaking Changes:**
- MessageBox namespace changed from `JEngine.Core.Misc` to `JEngine.UI`
- MessageBox moved to separate `jengine.ui` package (requires `openupm add com.jasonxudeveloper.jengine.ui` — [OpenUPM](https://openupm.com/packages/com.jasonxudeveloper.jengine.ui/))
- Bootstrap now uses `Prompt.ShowDialogAsync` delegate instead of direct `MessageBox.Show()` calls

**Packages:** Core v1.1.0, Util v1.1.0, UI v1.1.0
</Accordion>
</Accordions>
