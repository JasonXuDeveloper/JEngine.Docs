<Callout type="info" title="New in v1.1">
The `jengine.ui` package includes a full editor UI component library with a shadcn-inspired monochrome design system.
</Callout>

## Overview

JEngine UI provides a set of editor UI components built on Unity's UIToolkit (UI Elements). These are used internally by JEngine's editor panels (Bootstrap inspector, JEngine Panel) and are available for your own editor tools.

## Design System

The component library follows a **monochrome design** inspired by shadcn/ui:

- **True color inversion** between light and dark themes (dark Primary = light Secondary)
- **Design tokens** for consistent spacing, font sizes, border radius, and transitions
- **USS stylesheets** (`base.uss`, `components.uss`, `tokens.uss`) for theming

### Tokens

The `Tokens` class provides design constants:

| Category | Examples |
|----------|---------|
| **Colors** | Primary, Secondary, Muted, Border, Accent |
| **Spacing** | Xs (2px), Sm (4px), Md (8px), Lg (16px), Xl (24px) |
| **FontSize** | Xs (10px), Sm (12px), Base (14px), Lg (16px) |
| **BorderRadius** | Sm (4px), Md (6px), Lg (8px), Full (9999px) |

## JContainer

Added in v1.1.5, `JContainer` provides responsive max-width constraints following Tailwind/Bootstrap patterns:

```csharp
// Create a container with medium max-width
var container = new JContainer()
    .WithSize(ContainerSize.Md)       // 768px max-width
    .WithHorizontalPadding(Tokens.Spacing.Lg);

// Available sizes
public enum ContainerSize
{
    Xs = 480,
    Sm = 640,
    Md = 768,
    Lg = 1024,
    Xl = 1280,
    Full = 0   // No max-width limit
}
```

## Theming

The `JTheme` class applies consistent theming:

```csharp
// Apply theme to a root element
JTheme.Apply(rootVisualElement);
```
