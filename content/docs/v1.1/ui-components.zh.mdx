<Callout type="info" title="v1.1 新功能">
`jengine.ui` 包包含完整的编辑器 UI 组件库，采用 shadcn 风格的单色设计系统。
</Callout>

## 概述

JEngine UI 提供一套基于 Unity UIToolkit（UI Elements）构建的编辑器 UI 组件。这些组件在 JEngine 内部编辑器面板（Bootstrap 检视面板、JEngine Panel）中使用，也可用于你自己的编辑器工具。

### 依赖

- `com.cysharp.unitask` >= 2.5.10
- `com.unity.textmeshpro` >= 3.0.6

## 设计系统

组件库遵循 **单色设计** 风格，灵感来自 shadcn/ui：

- **真正的颜色反转** — 浅色主题的 Primary 在深色主题中变为 Secondary
- **设计令牌** — 统一的间距、字号、圆角和过渡
- **USS 样式表** — `base.uss`、`components.uss`、`tokens.uss`

### 令牌

`Tokens` 类提供设计常量：

| 类别 | 示例 |
|------|------|
| **颜色** | Primary、Secondary、Muted、Border、Accent |
| **间距** | Xs (2px)、Sm (4px)、Md (8px)、Lg (16px)、Xl (24px) |
| **字号** | Xs (10px)、Sm (12px)、Base (14px)、Lg (16px) |
| **圆角** | Sm (4px)、Md (6px)、Lg (8px)、Full (9999px) |

## JContainer

v1.1.5 新增，`JContainer` 提供响应式最大宽度约束，遵循 Tailwind/Bootstrap 模式：

```csharp
// 创建中等宽度的容器
var container = new JContainer()
    .WithSize(ContainerSize.Md)       // 768px 最大宽度
    .WithHorizontalPadding(Tokens.Spacing.Lg);

// 可用尺寸
public enum ContainerSize
{
    Xs = 480,
    Sm = 640,
    Md = 768,
    Lg = 1024,
    Xl = 1280,
    Full = 0   // 无最大宽度限制
}
```

## 主题

`JTheme` 类应用统一主题：

```csharp
// 对根元素应用主题
JTheme.Apply(rootVisualElement);
```

深色主题使用真正的灰度反转 — 浅色模式的 Primary 在深色模式变为 Secondary，反之亦然。
