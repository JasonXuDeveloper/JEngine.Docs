# JEngine v1.0 文档

欢迎使用 JEngine v1.0 文档！这个版本为 Unity 热更新开发引入了重大改进和新功能。

## v1.0 新特性

- 增强的性能和稳定性
- 改进的资源管理系统
- 更好地集成现代 Unity 功能
- 简化的开发工作流程
- 全面的测试框架
