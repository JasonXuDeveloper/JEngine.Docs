# 游戏出包须知

该文章将告诉您如何使用JEngine编译游戏包

[[toc]]



## 前言

上一篇文章，我们成功的对热更代码进行了配置，本文我们来打包游戏



## 出包步骤

- 确保**运行模式不是模拟模式**

- 进行**CLR绑定**（菜单栏，`JEngine/ILRuntime/CLR Binding`）

  > 不然会造成真机有代码被裁剪而无法进入热更代码

- 确保**在编辑器下使用对应模式能正常运行**

- 在`Build Settings`下，`Scenes In Build`不勾选热更场景

  > 不然会造成冗余，且会导致无法正常热更这些被勾选了的热更场景

- 出包！

- 测试打出来的包！
